## Quy tắc commit code:
Tên commit cần tuân theo quy tắc sau: `[FEATURE]: [Mô tả viết thường]`

Với FEATURE bao gồm:
- feat: Tính năg
- fix: Fix bug
- chore: nó tương đương với update của em, nhưng cái nhỏ
- refactor: chỉnh sửa, tối ưu code
- revert: revert code
- wip: đang làm chưa xong

Ví dụ: `feat: create new product`
